💡 Varför misslyckas hämtningen?
Appen öppnas troligen direkt från fil (file://), vilket blockeras av gratis CORS-proxys. Lösningarna i ordning från enklast:

1. Kör en lokal server (du har Python installerat?)
python -m http.server 8080  →  öppna http://localhost:8080/raceoffice.html

2. Lägg filen på din Nginx-server (Ikka)
Kopiera filen till /var/www/html/ och öppna via din DuckDNS-adress.

3. GitHub Pages — lägg filen i ett repo → Settings → Pages → gratis hostad URL.
